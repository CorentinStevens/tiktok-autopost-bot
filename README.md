# tiktok-autopost-bot

Python bot that automatically publishes videos from youtube links to tiktoks videos in many parts

## Getting Started
1) Download the repo localy :
`git clone https://github.com/CorentinStevens/tiktok-autopost-bot`

2) Install requirements :
`pip install -r requirement.txt` 

3) Change config in main.py

## Using

`python main.py`
